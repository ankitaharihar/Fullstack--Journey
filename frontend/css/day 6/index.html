Block level element

  • they occupy full width of their container
  • width is determined by content
  • height is determined by content
  • display:block
    • eg- div, p, h1, h2, h3, h4, h5, h6, ul, ol, li, header, section, footer, nav, main, article, aside

Inline level element

  • they only occupy as much width as necessary
  • width is determined by content
  • height is determined by content
  • display:inline
    • eg- a, span, strong, em, img, input, label, select, textarea, button

Inline-block level element

  • they only occupy as much width as necessary
  • width can be set by css
  • height can be set by css
  • display:inline-block
    • eg- elements that are set to display:inline-block in css
