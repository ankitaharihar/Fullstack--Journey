Features of JavaScript

  • Lightweight & Interpreted - executes line by line
  • Dynamic Typing - variables can change type
  • First-class Functions
  • Loosely Typed
  • Prototype-based Inheritance
  • Event-driven
  • Single-threaded
  • Supports Asynchronous Programming
  • Cross-platform
  • Rich Ecosystem
  • High-level Language

Difference: Java vs JavaScript

  • Java is statically typed; JavaScript is dynamically typed
  • Java is compiled; JavaScript is interpreted/JIT
  • Java is mainly server-side; JavaScript mainly client-side
  • Java syntax is more complex
  • Java focuses on OOP; JavaScript supports multiple paradigms

High-level vs Low-level Languages

  • High-level languages are human-friendly
  • Low-level languages are closer to machine code
  • High-level languages hide hardware details
  • Low-level languages offer more performance control
  • High-level -> application development, Low-level -> system programming

JIT (Just-In-Time Compilation)

JIT compilation improves JavaScript performance by compiling code into machine code at runtime. Frequently executed code paths are optimized for faster execution.
