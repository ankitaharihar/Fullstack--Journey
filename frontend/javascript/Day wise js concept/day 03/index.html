JavaScript Objects

An object is a collection of key-value pairs. It can contain properties and methods. Objects are mutable and stored by reference.

Objects in JavaScript

  • Access properties -> obj.key or obj["key"]
  • Add properties -> obj.key = value
  • Delete properties -> delete obj.key

Undefined and Null

  • undefined - variable declared but not assigned
  • null - intentional absence of value
  • typeof null returns "object" (known JavaScript bug)

Symbol

Symbols are a primitive data type introduced in ES6. Each symbol value is unique and immutable.

let vs const (Quick Notes)

  • const must be initialized and cannot be reassigned
  • let can be declared first and reassigned later

Operators in JavaScript

Arithmetic Special Cases

  • x / 0 -> Infinity
  • -x / 0 -> -Infinity
  • 0 / 0 -> NaN

Operator Types

  • Arithmetic: +, -, *, /, %
  • Assignment: =, +=, -=, *=, /=, %=
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !
  • Ternary: condition ? expr1 : expr2
  • typeof: typeof operator returns the type of a variable
  • increment/decrement: ++, --
