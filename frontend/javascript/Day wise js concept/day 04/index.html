Comparison Operators

  • lexicographical order - A-Z (Small), a-z (Large)
  • ASCII value - 65 (A), 97 (a)
  • == - checks equality of values
  • === - checks equality of values and data types
  • != - checks inequality of values
  • !== - checks inequality of values and data types

Logical Operators

  • && - logical AND - returns true if both operands are true
  • || - logical OR - returns true if at least one operand is true
  • ! - logical NOT - returns true if the operand is false

Assignment Operators

  • compound assignment operators - shorthand for performing an operation and assignment in one step
  • += - adds a value to a variable and assigns the result to that variable
  • -= - subtracts a value from a variable and assigns the result to that variable
  • *= - multiplies a variable by a value and assigns the result to that variable
  • /= - divides a variable by a value and assigns the result to that variable
  • %= - calculates the remainder of dividing a variable by a value and assigns the result to that variable

Pre-increment and Post-increment

  • pre-increment (++variable) - increments the variable before using its value
  • post-increment (variable++) - uses the variable's value before incrementing it

Pre-decrement and Post-decrement

  • pre-decrement (--variable) - decrements the variable before using its value
  • post-decrement (variable--) - uses the variable's value before decrementing it

Ternary Operator

  • condition ? expression1 : expression2 - evaluates the condition and returns expression1 if true, otherwise returns expression2
