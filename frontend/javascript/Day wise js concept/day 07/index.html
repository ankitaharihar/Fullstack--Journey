Functions in JavaScript

Predefined Functions

Functions which are already defined (pre-written), for example:

  • console.log() — prints output in console (returns undefined)
  • Math.max() — returns maximum value
  • Math.min() — returns minimum value
  • Math.floor() — returns floor value
  • Math.ceil() — returns ceil value
  • Math.random() — returns random value between 0 and 1, 1 is excluded
  • Math.sqrt() — returns square root

Browser Functions

  • alert() — shows alert message (example: alert("Good Morning")) — returns undefined
  • prompt() — shows input box (example: prompt("Enter your name"))
  • confirm() — shows confirm box — returns true/false (example: confirm("Are you sure?"))

  1. Accept a number from user and check even or odd
  2. Take two numbers from user and display their sum
