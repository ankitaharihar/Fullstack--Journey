How to Select Elements in the DOM

why we should not use innerHTML because it is vulnerable to XSS attacks, it can

  • i] id - will return you element with given id if there is no element with that id, it returns null [getElementsByID(id name)]

  • ii] class - will return you a HTML collection of elements with given class name if there is no element with that class name, it returns empty HTML collection - array like object [getElementsByClassName(class name)]

  • iii] tag name - will return you a HTML collection of elements with given tag name if there is no element with that tag name, it returns empty HTML collection [getElementsByTagName(tag name)]

  • iv] query selector - will return you first element that matches the given CSS selector if there is no element that matches the given CSS selector, it returns null [querySelector(CSS selector)]

  • v] query selector all - will return you a NodeList of elements that matches the given CSS selector if there is no element that matches the given CSS selector, it returns empty NodeList [querySelectorAll(CSS selector)]
