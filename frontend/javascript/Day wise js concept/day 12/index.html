Events and Events Handling

What is an Event? An event is an action that happens on a webpage. It can be caused by: a user the browser JavaScript itself Examples of events: Event What happens click user clicks a button mouseover mouse goes over element keydown keyboard key pressed submit form submitted scroll page scroll


So basically: Event = something that happens on the page.

Examples

Event Handling

Event handling is the process of responding to events that occur on a webpage.

Event Listeners

An event listener is a function that waits for an event to occur and then executes a block of code when the event is detected.

Event Propagation

Event propagation is the process by which an event travels through the DOM tree. There are two phases: capturing and bubbling.

Event Objects

An event object is created when an event occurs. It contains information about the event, such as the type of event, the element that triggered the event, and any data associated with the event.

Event Delegation

Event delegation is a technique where an event listener is attached to a parent element, and it handles events for its child elements. This can be useful for dynamically added elements or when you want to reduce the number of event listeners.

Event Bubbling

Event bubbling is the default behavior of event propagation, where an event triggered on a child element will bubble up to its parent elements.

3 ways to handle events

1. Inline event handling: This involves adding an event handler directly to an HTML element using the "on" attribute. For example: <button onclick="alert('Button clicked!')">Click me</button>

2. DOM Level 0 event handling: This involves assigning a function to an event property of an HTML element. For example: const button = document.querySelector('button'); button.onclick = function() { alert('Button clicked!'); };

3. DOM Level 2 event handling: This involves using the addEventListener method to attach an event listener to an HTML element. For example: const button = document.querySelector('button'); button.addEventListener('click', function() { alert('Button clicked!'); });
