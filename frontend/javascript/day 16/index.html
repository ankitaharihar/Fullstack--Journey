Summary

  • Call stack - callstack is a data structure that keeps track of function calls in JavaScript
  • Event loop - event loop is a mechanism that allows JavaScript to perform non-blocking operations by offloading tasks to the browser or Node.js
  • Asynchronous operations - asynchronous operations are tasks that can be performed without blocking the main thread of execution, such as network requests or timers
  • Callbacks - callbacks are functions that are passed as arguments to other functions and are executed after the asynchronous operation is completed
  • Promises - promises are objects that represent the eventual completion (or failure) of an asynchronous operation and its resulting value
  • Async/Await - async/await is a syntax for writing asynchronous code that is easier to read and understand than using callbacks or promises
