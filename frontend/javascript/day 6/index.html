Loops in JavaScript

Rules for Loops

  • Never write infinite loops accidentally (e.g., for(;;))
  • Always use break and continue with proper conditions
  • Make sure loop condition eventually becomes false

Types of Loops

  • for loop
  • while loop
  • do...while loop

for Loop

when you know exact number of iterations

Syntax:

for (initialization; condition; increment/decrement) {
  // code to be executed
}

while Loop

When you dont know the exact no of iterations

Syntax:

while (condition) {
  // code
}

do...while Loop

When you want to exacute at least once time to run code

Syntax:

do {
  // code
} while (condition);
✅ for loop → when iterations known ✅ while loop → when condition-based ✅ do…while → runs at least once
