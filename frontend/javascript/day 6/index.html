Loops in JavaScript

Example: Given a number, check whether it is prime or not. If prime → print "Number is prime" Else → print "Number is not prime"

Rules for Loops

  • Never write infinite loops accidentally (e.g., for(;;))
  • Always use break and continue with proper conditions
  • Make sure loop condition eventually becomes false

Types of Loops

  • for loop
  • while loop
  • do...while loop

for Loop

Syntax:

for (initialization; condition; increment/decrement) {
  // code to be executed
}
---

Example 1 — Print 1 to 10

for (let i = 1; i <= 10; i++) {
  console.log(i);
}

Output: 1 2 3 ... 10

Example 2 — Prime Number Check

let num = 7;
let isPrime = true;

if (num <= 1) {
  isPrime = false;
} else {
  for (let i = 2; i < num; i++) {
    if (num % i === 0) {
      isPrime = false;
      break;
    }
  }
}

if (isPrime) {
  console.log("Given number is prime");
} else {
  console.log("Given number is not prime");
}

while Loop

Syntax:

while (condition) {
  // code
}

Example — Print 1 to 5

let i = 1;

while (i <= 5) {
  console.log(i);
  i++;
}

do...while Loop

Syntax:

do {
  // code
} while (condition);

Example — Runs at least once

let i = 1;

do {
  console.log(i);
  i++;
} while (i <= 5);
✅ for loop → when iterations known ✅ while loop → when condition-based ✅ do…while → runs at least once
